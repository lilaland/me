## Software Engineering Intern, Cloud Insights and Telemetry
**Summer 2025**

Worked within the Cloud Insights and Telemetry team, contributing to the infrastructure that gives MongoDB visibility into how its managed deployments behave at scale.

- Extended the MongoDB Agent — written in Go — with a new data collector that queries active deployment metadata, enabling richer telemetry signals across the managed fleet
- Built a Java control plane endpoint responsible for processing, tokenizing, and streaming collected deployment data to AWS Kinesis for downstream consumption
- Ensured end-to-end pipeline stability with a full test suite covering unit, integration, and E2E scenarios — giving the team confidence to ship without regressions
- Navigated a large, mature Go and Java codebase with established patterns, learning to contribute meaningfully to a production system used across MongoDB Atlas infrastructure
- Collaborated with a distributed team across planning, design review, and code review cycles typical of a professional engineering org

**Stack:** Go · Java · AWS Kinesis · Unit / Integration / E2E testing
